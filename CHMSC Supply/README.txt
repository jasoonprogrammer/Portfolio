Step 1: Install Python 3.8 (Check Add to PATH)
Step 2: run Command Prompt or cmd and change directory to this folder
Step 3: type the following command

python -m pip install --upgrade pip
python -m pip install requirements.txt

NOTE: requirements.txt that is found in this folder, locate it.

Step 4: Once everything required is installed. run the following commands

cd newproject
manage.py runserver

If everything goes well. It should run the local server.
then go to this url.

localhost:8000/app/home
